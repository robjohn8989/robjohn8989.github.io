Ostrij
About

Ostrij is an innovative Salesforce integration tool, which was hatched with an idea to solve the problem of connecting multiple applications to Salesforce. It helps the users/companies to keep their data in sync with various tools like Slack, Shiftbord, WorkFront, JIRA, Quickbooks, etc. over Salesforce.

Implementation

Designed a brand new digital catalog, implemented a site-wide call to action, and implemented a brand new solution in line with the current site design. Implemented the software architecture for the user to subscribe simply to the services available and create various type of flows to sync all the data (Data Sync Flows) or to sync the data based on certain conditions.

Technology Stack

Language: Java, JavaScript, TypeScript, HTML, CSS
Framework: Spring Boot, React, Next, Tailwind, Node
Database: PostgreSQL
Server: Ubuntu Apache
Others: SSL
