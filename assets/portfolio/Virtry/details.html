Virtry
About

VirTry is a trial based unique service platform that enables the users to virtually try the headphones and share best of their looks on social media platforms. Users can browse through a wide variety of headphones and click on the preferred one to select it for trial. They can also upload the pictures to the database and replace the current one with any of them when needed. This application also allows the users to edit the image before sharing it to give it the desired touch, and even request admins to add products to the application.

Implementation

Implemented the website that comes with three roles based user login 'Business User,' 'User' and 'Admin.' Upon login, users will be redirected to the product page. Also provided the function that allows users to edit and customize headphones according to images. Used a query-based framework for supporting responsive modes on various web and mobile devices.

Technology Stack

Language: Python, SQL, HTML5, CSS, JavaScript, TypeScript
Framework: Django, React, Web3, Node, Redux, Tailwind, jQuery, FabricJs, Bootstrap
Database: MySQL
Payment Gatway: Stripe
Server: Ubuntu Apache
Web Services: Python RESTful API
