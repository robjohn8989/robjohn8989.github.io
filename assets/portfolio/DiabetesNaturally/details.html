DiabetesNaturally
About

Diabetes Naturally is a healthcare web development and mobile application dedicated to support diabetic patients at different stages. It helps in managing diabetes by letting one track his insulin or blood glucose level for easy access to his concerned medical professional.

Implementation

Built it with a query-based framework for supporting responsive modes on various web and mobile devices. Implemented many useful functions such as setting their goals, manage food intake, keep a check on calories burnt, set remainder for medication, track blood glucose level, weight management, stress management and keep notes on their health improvement. Also included a forum for paid users enabling them to ask health-related questions to experts on the app.

Technology Stack

Language: C#, Javascript, TypeScript, HTML, CSS, SQL
Framework: ASP.NET MVC, React, Node, Tailwind
Database: PostgreSQL
Development Tool: Visual Studio, VSCode
Web Services: ASP.NET Web API
Server: Azure
