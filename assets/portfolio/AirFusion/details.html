AirFusion
About

Air Fusion is an AI-based turbine management system that functions across the USA. The solution works seamlessly to offer the benefits of reduced maintenance cost and decreased risk to human resource. It integrates with the well-managed system of drones that captures images of the turbines and uploads them on the server for the analysts to view.

Implementation

Built an intuitive user interface to analyze the images to identify the spots of damage which are compiled into a report. Used Computer Vision algorithm to detect automatically the spots on the turbine.

Technology Stack

Language: C#, Javascript, TypeScript, HTML, CSS, SQL
Framework: ASP.NET MVC, Angular, Node
Database: SQL Server
Development Tool: Visual Studio, VSCode
Web Services: WEB API
Server: Azure
Others: Computer Vision, Image Processing, Machine Learning
