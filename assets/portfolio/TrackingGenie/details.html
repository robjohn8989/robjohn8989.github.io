Tracking Genie
About

Tracking Genie, as the name suggests, is a smart, GPS-based solution for tracking vehicles. Using a combination of closely integrated hardware and software, it provides the minutest details about a vehicle's location as well as its overall usage - real-time! It is a perfect combination of server-side application, hardware devices that collect the data, and efficient customer-facing web and mobile applications.

Implementation

Tried to implement a wide array of benefits including easy installation, user-friendly features, and comprehensive details on fingertips, competitive price, and 24*7 assistance. In other words, designed the software architecture to function across the vehicles and mobile devices and to extend your visibility inside as well as away from the vehicle. Utilized the Digital Signal Processing and modern industrial technologies.

Technology Stack

Language: Python, SQL, HTML, CSS, JavaScript, TypeScript, Java, Swift
Framework: Django, React, Node, Tailwind, jQuery
Database: Postgre
Server: AWS
Web Services: Python RESTful API
Others: Google Maps, CC Avenue Payments, Bulk email services, Bulk SMS services
